# Deployment

Deployments are performed using Wrangler.

Development flow:

1. Create feature branch
2. Make changes
3. Open Pull Request
4. Merge into main
5. Deploy

Future deployments should be automated using GitHub Actions.

---

## Development

Deploy to development environment first.

---

## Production

Production deployments occur only after testing.

---

## Rollback

Every deployment should be reversible.
