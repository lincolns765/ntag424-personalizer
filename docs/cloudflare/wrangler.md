# Wrangler

Wrangler is the official Cloudflare CLI.

It is used to:

- Authenticate
- Create Workers
- Create D1 databases
- Create R2 buckets
- Create KV namespaces
- Deploy Workers
- Manage secrets

---

## Typical Commands

Login

wrangler login

Deploy

wrangler deploy

Create D1

wrangler d1 create

Create R2

wrangler r2 bucket create

Create KV

wrangler kv namespace create
