# 

# \# Goal

# 

# Replace the Jupyter notebook as the primary user interface with a desktop dashboard built using PySide6.

# 

# The dashboard will become the operational control center for the Collectible Authentication Platform.

# 

# The existing ingestion modules developed in PR-005 (folder scanner, image pairer, pipeline, etc.) must be reused. No business logic should exist inside the GUI.

# 

# This PR establishes the application framework only. It does not perform uploads, AI processing, or database writes.
