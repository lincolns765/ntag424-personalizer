# \# Goal

# 

# Implement the first functional workflow of the desktop application.

# 

# This PR connects the existing ingestion pipeline to the desktop GUI, allowing users to scan an image folder, pair front/back photos, and review the results before any files are renamed, uploaded, or written to the database.

# 

# No Cloudflare uploads, database writes, GUID generation, or AI processing will occur in this PR.
